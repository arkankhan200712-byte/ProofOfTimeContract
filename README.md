# SafeRent Contract

## Project Title
SafeRent Contract

## Project Description
SafeRent Contract is a smart contract-powered platform securing property rental agreements on blockchain. Built with Soroban smart contracts on Stellar, it automates rental term enforcement, rent payments, security deposits, and agreement lifecycle management ensuring transparency and trust between landlords and tenants.

## Project Vision
This project aims to disrupt traditional rental management by embedding lease terms and payment schedules into verifiable, immutable smart contracts that eliminate disputes, reduce intermediaries, and promote prompt payments and compliance.

## Key Features
- Rental Agreement Creation: Landlords define rental terms including rent, deposit, and duration.
- Automated Payments: Enforces rent payment schedules and due dates.
- Agreement Status: Tracks active, terminated, or expired lease states.
- Immutable Contracts: Transparent, tamper-proof storage on blockchain.
- Secure Roles: Restricted actions enforced by tenant and landlord roles.
- Real-time Queries: Access detailed lease info and payment history.
- Lifecycle Management: Supports rental termination and expiry.

## Usage Instructions
1. Landlord creates detailed rental agreements on blockchain.
2. Tenants pay rent when due using contract function.
3. Landlords can terminate agreements securely.
4. All parties query rental status and history at any time.

## Future Scope
- Escrow and dispute resolution features.
- Automated renewal and lease renegotiation contracts.
- Integration with payment gateways for tokenized rent.
- Support for multi-tenant and subleasing agreements.
- Mobile apps for lease management and notifications.
- Legal compliance and smart contract audits.

## Technology Stack
- Soroban Rust SDK enabling secure and transparent smart contracts.
- Stellar blockchain offering decentralization and immutable records.
- Cryptographic identity and authorization enforcement.
